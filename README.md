# whCheckJar

Безопасный аудит каталога `plugins` для Paper 1.16.5.

Плагин:

- читает `plugin.yml` из каждого JAR в корне `plugins`;
- показывает, загружен и включён ли найденный плагин;
- сообщает о повреждённых JAR и JAR без `plugin.yml`;
- ищет каталоги, которым не соответствует ни один найденный плагин;
- создаёт подробные отчёты в `plugins/whCheckJar/logs`;
- **никогда не удаляет и не перемещает файлы или папки**.

## Сборка

Требуется JDK 8 или новее и Maven:

```bash
mvn clean package
```

Готовый файл: `target/whCheckJar-1.0.0.jar`.

## Команды

- `/whcheckjar scan` — запустить проверку;
- `/whcheckjar status` — повторно показать итог последней проверки;
- `/whcheckjar reload` — перезагрузить `config.yml`.

Псевдоним команды: `/wcj`. Право: `whcheckjar.admin` (по умолчанию только OP).

## Важно

Некоторые плагины намеренно создают дополнительные общие папки. Такие папки нужно
добавить в `ignored-folders` в `config.yml`. Поэтому результат называется списком
подозрительных папок, а решение о ручном удалении всегда принимает администратор.
